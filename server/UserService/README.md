# User Micro service

## packages installed

npm i -S @nestjs/microservices @nestjs/mongoose bcrypt class-validator
npm i -D @types/bcrypt

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
